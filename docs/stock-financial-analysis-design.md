# 股票财报分析设计

本文档定义股票财报与公司质量分析的第一版设计。该能力只适用于 `STOCK`，不适用于 `ETF`、`LOF` 或 `FUND`。

## 目标

股票分析要从“价格、趋势、行业、风险”扩展到“公司质量、财报趋势、估值位置和财报事件”。财报不是股票分析的全部，但它是公司质量和估值判断的核心输入。

第一版要回答：

- 公司最近几个季度质量是变好还是变差？
- 收入、利润和现金流是否匹配？
- 负债、应收、存货、商誉是否出现异常？
- 当前估值相对历史和行业是否偏高？
- 财报变化是否应触发风险事件或重要事项？
- `fundamental_score`、`valuation_score` 和 `risk_score` 的依据是什么？

## 适用边界

- 只处理 `asset_type = STOCK` 的资产。
- ETF、LOF、场外基金不进入股票财报分析。
- AI 只解释财报数据、评分依据和规则结果，不直接根据财报自由生成买卖建议。
- 财报数据缺失时不能生成确定性公司质量结论。
- 样本数据、估算数据和真实披露数据必须区分展示。

## 数据模型规划

建议新增以下数据层：

- `financial_statement_snapshot`：财报原始口径快照，覆盖利润表、资产负债表、现金流量表、分红和回购。
- `financial_metric_snapshot`：财务指标快照，覆盖营收增速、净利润增速、毛利率、净利率、ROE、ROA、负债率、经营现金流和自由现金流。
- `valuation_snapshot`：估值快照，覆盖 PE、PB、PS、股息率、历史分位、行业分位和估值区间。
- `financial_event`：财报事件，覆盖业绩预告、定期报告、利润大幅变化、现金流异常、资产减值、审计意见和管理层变化。
- `stock_quality_snapshot`：公司质量快照，汇总成长、盈利、现金流、资产质量、负债、估值和风险结论。

所有快照都应至少包含：

- `symbol`
- `name`
- `report_period`
- `report_type`
- `data_date`
- `source`
- `source_mode`
- `data_version`
- `created_at`

`source_mode` 建议区分：

- `REAL`：真实披露或可信数据源。
- `ESTIMATED`：估算或推导数据。
- `SAMPLE`：样本数据。
- `MISSING`：缺失占位。

## 指标口径

第一版指标保持可解释，不追求复杂因子模型。

公司质量：

- 成长：营收增速、净利润增速、扣非净利润增速。
- 盈利：毛利率、净利率、ROE、ROA。
- 现金流：经营现金流、自由现金流、经营现金流 / 净利润。
- 资产质量：应收账款增速、存货增速、商誉占比。
- 负债：资产负债率、有息负债率、利息保障倍数。

估值：

- 当前 PE / PB / PS。
- 近 3 年 / 5 年历史分位。
- 行业分位。
- 股息率和分红稳定性。

风险事件：

- 利润增长但现金流恶化。
- 收入下滑或利润大幅下滑。
- 毛利率或净利率连续下滑。
- 负债率明显上升。
- 应收、存货、商誉异常增长。
- 估值高于历史高分位。
- 审计意见、减值、管理层变化等事件。

## Worker 流程

建议新增或扩展以下步骤：

1. 同步财报和估值基础数据。
2. 标记数据来源和数据日期。
3. 计算财务指标快照。
4. 计算估值快照。
5. 识别财报事件和异常。
6. 生成股票质量快照。
7. 将异常写入 `risk_event`。
8. 将重要异常接入 `review_task`。
9. 更新股票分析、日报和 AI 解释输入。

该流程应在股票分析之前完成，避免股票分析使用过期或缺失的财务指标。

## API 和页面范围

API 第一版建议提供：

- 股票财报摘要。
- 财务指标趋势。
- 估值分位。
- 财报事件列表。
- 股票质量快照。

页面第一版建议增加到股票分析页，而不是新增复杂财报终端：

- 公司质量摘要。
- 近几个季度核心指标趋势。
- 当前估值位置。
- 财报事件和风险提示。
- 数据来源和数据日期。
- AI 财报解释。

## 复盘闭链接入

财报异常不应该只停留在股票页面，应进入复盘闭环：

- 写入 `risk_event`：记录异常类型、严重程度、数据日期和说明。
- 生成 `review_task`：只对重要异常生成事项，避免刷屏。
- 写入日报：解释今天哪些股票因财报变化需要关注。
- 进入 AI：解释财报变化、指标变化和建议依据。
- 影响建议：只通过规则影响分级建议，AI 不直接生成建议。

## 验收标准

- 股票基本面评分能追溯到指标、数据日期和数据来源。
- 估值判断能说明当前估值相对历史和行业的位置。
- 财报异常能生成风险事件或重要事项。
- 缺失、样本、估算和真实数据能被区分。
- ETF、LOF、FUND 不受股票财报口径影响。
