# P0-003: 引入资产类型字段

- Status: DONE
- Priority: P0
- Owner: Codex
- Created At: 2026-06-20
- Completed At: 2026-06-20

## Goal

股票、ETF、场外基金在系统内有明确资产类型，避免观察池、持仓和策略信号都被当成普通股票。

## Scope

本任务只完成资产类型字段、推断、回填和页面展示：

- STOCK：股票。
- ETF：场内 ETF / LOF。
- FUND：场外基金。

基金净值、基金分析和基金独立页面属于后续 P1 任务。

## Changes

- 观察池、持仓、策略信号增加资产类型字段。
- 初始化脚本支持旧库自动补列和回填。
- 后端服务支持资产类型推断。
- worker 生成信号时写入资产类型。
- 观察池、持仓、信号页展示资产类型。

## Acceptance

- 股票、ETF、基金能区分展示。
- 旧库重新初始化后自动补齐字段。
- ETF 代码能被识别为 ETF。
- 信号返回资产类型。
